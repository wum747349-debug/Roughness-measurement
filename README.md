# 粗糙度测量系统 (Roughness Measurement System)

这是一个完整的分布式测量与控制系统，由基于 C# 的上位机软件和基于 STM32F103C8T6 的 24V 工业级下位机协同工作。

## 系统架构与目录说明

*   **Firmware/**: 包含 STM32 下位机核心 C 代码。
    *   主要负责 24V 逻辑电平下控制步进电机、处理限位传感器输入。
    *   通过中断高频采集传感器实时数据，并按自定义二进制帧协议向PC发送串口流。

*   **Software/**: 包含基于 C# WinForms (`.NET Framework` / `.NET Desktop`) 构建的上位机软件代码。
    *   负责波特率与通信参数配置。
    *   将收到的二进制数据转换处理，应用多阶去趋势滤波（如高通分离、鲁棒 LOESS）。
    *   计算并图形化展示 Abbott-Firestone 曲线及 7 个核心粗糙度参数（Ra, Rz, Mr1, Mr2, Rpk, Rvk, Rk）。
    *   集成 SQLite 将每次测量的完整波形与计算结果持久化，支持历史查询和离线回放。
